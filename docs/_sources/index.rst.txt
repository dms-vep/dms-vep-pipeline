For the actual code, see https://github.com/dms-vep/dms-vep-pipeline

Data files
----------
- `parental gene sequence <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/gene_sequence/codon.fasta>`_
- `parental protein sequence <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/gene_sequence/protein.fasta>`_
- `sequential-to-reference site numbers <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/data/site_numbering_map.csv>`_
- `codon-variant table <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/variants/codon_variants.csv>`_
- `processed barcode sequencing runs <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/barcode_runs/processed_barcode_runs.csv>`_
- `variant counts <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/variant_counts>`_
- `functional selection experiments <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/func_scores/functional_selections.csv>`_
- `mutation effects for each functional selection <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/globalepistasis_fits>`_
- `mutation functional effects replicate average (observed phenotype) <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/muteffects_functional/muteffects_observed.csv>`_
- `mutation functional effects replicate average (latent phenotype) <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/muteffects_functional/muteffects_latent.csv>`_
- `antibody selection experiments <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/prob_escape/antibody_selections.csv>`_
- `antibody escape values <https://github.com/dms-vep/dms-vep-pipeline/blob/main/test_example/results/antibody_escape>`_
